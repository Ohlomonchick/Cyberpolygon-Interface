Мониторинг прогресса участников

  1. В таблице можно наблюдать за прогрессом участников. В каждой строке отражены данные об участнике (студенте) и сколько он решил заданий.
  2. Рядом с таблицей есть таймер, который показывает, сколько времени осталось до окончания работы.
  3. Под таблицей есть кнопка "Закончить сейчас". Она завершает работу для всех участников.
  4. После завершения работу можно продлить на 15 минут.

Шаги создания лабораторной работы

  1. Откройте страницу "Лабораторные работы".
  2. Выберите программу.
  3. Выберите лабораторную работу, нажав на тип занятия.
  4. Выберите задания, которые будут выданы студентам (по умолчанию выбраны все).
  5. Выделите время на работу (по умолчанию - неделя) и нажмите на кнопку "Создать".
  6. Вас перебросит на страницу работы, где можно наблюдать за прогрессом участников.
